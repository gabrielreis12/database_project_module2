O arquivo EER Diagram.pdf possui o diagrama DER das entidades das tabelas feito no MySQL Workbench (modelagem lógica);

O arquivo conceptual_model.png possui a modelagem conceitual feita no BRMODELOWEB;

O arquivo SQL_code_module2_ExercicioEscola.sql possui o script para a criação das tabelas do arquivo EER Diagram.pdf;

Esse repositório serviu como um exercício que estava fazendo de como passar parâmetros de um CRUD em SQLite3 com kwargs e **args, mas com a finalidade de adiantar a entrega 
não apliquei essa lógica a todo o código. Desta forma, o repositório serviu para cumprir os requisitos de entrega do Módulo 2 e do Módulo 8 para obtenção de nota parcial na
aprovação da fase 1 do programa Bolsa Futuro Digital.
